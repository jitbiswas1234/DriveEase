# 🚗 DriveEase - Car Rental Management System

DriveEase is a web-based Car Rental Management System developed using PHP and MySQL. The platform allows users to browse available cars, book vehicles online, make payments, manage bookings, and download invoices. Administrators can manage cars, customers, bookings, payments, and generate reports through a dedicated dashboard.

## 📌 Features

### User Features

* User Registration & Login
* Browse Available Cars
* View Car Details
* Book Cars Online
* Secure Payment Integration
* Booking History
* Download PDF Invoices
* Profile Management
* Cancel Bookings

### Admin Features

* Admin Dashboard
* Add, Edit & Delete Cars
* Manage Customers
* Manage Bookings
* Payment Management
* View Contact Messages
* Generate Reports
* Dashboard Statistics
